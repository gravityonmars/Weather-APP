
# Weather Web App

A simple and responsive weather web application built using HTML, CSS, and JavaScript.  
This app fetches real-time weather data using the WeatherAPI service based on the city name entered by the user.

## Features

- Search weather by city name
- Displays temperature in Celsius
- Shows weather condition, wind speed, and humidity
- Uses real-time data from WeatherAPI
- Clean and beginner-friendly user interface

## Technologies Used

- HTML
- CSS
- JavaScript
- WeatherAPI

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/weather-web-app.git
````

### 2. Get an API Key

* Visit [https://www.weatherapi.com](https://www.weatherapi.com)
* Create a free account
* Copy your API key

### 3. Configure API Key

Open `script.js` and replace:

```javascript
const apiKey = "YOUR_WEATHERAPI_KEY";
```

with your actual API key.

### 4. Run the Project

Simply open `index.html` in your web browser.
